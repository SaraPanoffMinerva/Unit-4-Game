Puddle Jumper Game

Rules of the Game

  • You will be given a random number at the start of the game.
  • There are four Puddle Jumper Cards. By clicking on a card you will add amounts to your score.
  • To win the game your total score must match the random number given.
  • Each card's value will be hidden until you click on it
  • Each time the game starts, the cards values will change
